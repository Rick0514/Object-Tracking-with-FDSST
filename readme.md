## Object tracking applying the FDSST



> 机器人创新实践课的一个子程序，较为稳定的物体跟踪代码



### 平台

- [x] win10
- [x] VS2017
- [x] opencv3


### 原功能


- [x] 跟踪框出的物体
- [x] 自适应物体大小  


### 新增功能

- [x] 物体离开视野后再回来可跟踪
- [x] 修复了一个源程序的bug：有时出视野后再进入会退出
- [x] 增加了串口通信，可把跟踪物体的信息通过串口发送出去
